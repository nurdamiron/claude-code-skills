# Каталог скиллов по нишам

Проверено: **2026-05-31** (звёзды — через GitHub API на эту дату). Все ссылки — живые репозитории.

> Большинство репо — для **Claude Code** (терминал). В Claude.ai (чат) их можно использовать, упаковав нужную папку `skill/` в `.zip` и загрузив через Settings → Capabilities.
>
> Команды установки у авторов разные: где-то `/plugin marketplace add`, где-то `npx skills add`, где-то просто скопировать `SKILL.md` в `~/.claude/skills/`. Смотри README конкретного репо.

## ⚠️ Заметки

- Звёзды и состав репо меняются — числа приблизительные на дату проверки в шапке.
- У популярных репо бывают форки-клоны с тем же описанием (например у `coreyhaines31/marketingskills` — `ayrshare/marketingskills`, `syntax-syndicate/marketing-skills`) — ставь **оригинал**.
- Community-скилл — это чужие инструкции, которые Claude будет исполнять у тебя. Перед установкой открой `SKILL.md` и доверяй проверенным авторам. В security-списках периодически вычищают фишинговые ссылки — слепо доверять каталогу нельзя.
